## 핵심 구성
- Docker 기반 컨테이너화
- Jenkins CI/CD 파이프라인
- Redis 캐싱 및 JWT 관리
- ELK Stack 로그 분석
- 컨테이너 기반 확장 가능 구조

## 주요 기술 특징
1. **확장성**
- Docker 컨테이너화
- 컨테이너 기반 설계
- Jenkins CI/CD

2. **AI 기술 통합**
- DeepFace 얼굴 인식
- Upstage OCR
- LLaMA 3.2 챗봇

3. **실시간 처리**
- WebRTC P2P 연결
- Redis 캐싱
- WebSocket 실시간 통신

4. **보안 및 모니터링**
- JWT 기반 인증
- ELK Stack 로그 분석
- Redis 토큰 관리
